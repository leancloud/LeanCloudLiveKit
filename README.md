# LeanCloudLiveKit

LeanCloudLiveKit 是 LeanCloud 与七牛合作推出来的直播＋IM SDK，其功能包括：

1. 直播与观看直播
1. 直播中的聊天系统
1. 弹幕
1. 打赏和礼物

可以参考以下示例:
